# Survey Poll Bot

A Python-based survey and polling bot that supports multiple question types 
(MCQ, rating, and text responses) with analytics and data export features.

## Features

- Multiple choice questions (MCQ)
- Rating questions
- Text responses
- Survey analytics
- Data export
- SQLite database

## Project Structure

bot.py           # Main bot logic
database.py      # Database connection
models.py        # Data models
analytics.py     # Survey analytics
export.py        # Export results
views/           # Survey question types
